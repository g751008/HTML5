網際網路(Internet) = 把家庭內的少人數網路和企業使用的大規模網路等不同種類網路互相連結起來的世界級網路，以一組標準的網路TCP/IP協定族相連。

WWW(World Wide Web) = 其中一種存取網際網路上的資訊的機制，只要存取畫面上的超連結就能前往網路上的各種地方(此機制稱為超文件系統 Hypertext System)。

HTML(Hyper Text Markup Language) = 用來寫網頁的標籤語言，以<>標籤框住來表現區塊。
1990由隸屬CERN(歐洲核子研究機構)的Tim Berners-Lee開發，同一時期HTTP、URI、WWW、世上第一個Web瀏覽器也被開發出來。當時被用於CERN內部共享研究成果，1993年CERN正式發表WWW並推動網際網路連線服務後，被確立為網際網路上的文件標準規範。
HTML最初為研究者專用，1993年隸屬於伊利諾大學的NCSA(國家超級電腦應用中心)的Mark Anderson開發的Web瀏覽器Mosaic能同時顯示文字與圖片，遠比以往Web瀏覽器方便，在此之後登場的Netscape Navigator和Internet Explorer，讓網際網路在一般使用者之前普及。
HTML最早由IETF(Internet Engineering Task Force)制定標準，3.2後由Tim Berners- Lee組建的W3C負責。1993:HTML1.0 1995:HTML2.0 1997:HTML3.2 1997:HTML4.0 1999:HTML4.1。 之後W3C制定XHTML來取代HTML。但制定2.0版前宣告終止，而將心力放在制定HTML5上。

Web伺服器 = 負責回應Web瀏覽器要求(Request)並提供服務回應(Response)的電腦，藉由HTTP(Hyper Text Transfer Protocol)和Web瀏覽器進行通訊，除傳送HTML網頁也可傳送相關圖片或音訊內容。

動態網頁 = 可依用戶端要求執行程式並傳送結果的網頁(另一方面，只能傳回固定內容則稱靜態網頁)。

CGI(Common Gateway Interface) = 依照來自Web瀏覽器的要求啟動Web伺服器上的程式的機制，用於CGI的程式是以Perl或C語言之類的程式語言所寫成。

伺服器端腳本(Server Side Script) = 先在Web伺服器端的HTML檔案中寫上腳本(Script;簡單的程式)，再將執行結果作為HTML一部分傳回給Web瀏覽器，此為伺服器端腳本。

用戶端腳本 = 相對於伺服器端腳本，在Web瀏覽器上執行的腳本。主流為JavaScript。

HTML5 = W3C(World Wide Web Consortium)制定的HTML新標準，包含CSS3(Cascading Style Sheets 3)和JavaScript API(Application Programming Interface)。

IE (Internet Explorer) = 微軟開發的Web瀏覽器，隨附於Windows。

Google Chrome = Google開發的Web瀏覽器，可與Google服務接軌。

Mozilla Firefox = Mozilla foundation開發的Web瀏覽器，有強大的擴充性。

Opera = Opera Software開發的Web瀏覽器，有豐富的功能。

Safari = 蘋果公司開發的Web瀏覽器，能與Mac、iPhone等Apple產品接軌。

AJAX = Asynchronous JavaScript + XML(非同步JavaScript + XML)的簡稱，是JavaScript融合CSS、HTML和XML。原只是Google Map使用的技巧，2005年Jesse James Garrett發表的文章正式出現AJAX這個名詞。 關鍵是藉由XMLHttpRequest這個JavaScript物件與伺服器進行非同步通訊。
